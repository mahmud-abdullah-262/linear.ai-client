Project Overview & Architecture:
Linear.ai is an advanced Agile Project Management tool integrated with an Autonomous AI Co-Pilot. It streamlines task workflows through an interactive Kanban board and leverages Agentic AI to automatically categorize tasks and answer contextual, cross-project questions.

Tech Stack BlueprintFrontend: Next.js (App Router), TypeScript, Tailwind CSS, TanStack Query (for server state & cache invalidation), Recharts (for analytics).Backend: Node.js, Express.js, TypeScript, MongoDB (Atlas).

Authentication: Better Auth + Google OAuth.

AI Engine: Google Gemini 2.5 Flash / Groq (Llama 3) via official SDKs.2.
